# Mars Syntax App

**GitHub Repository:** [Mars Syntax App](https://github.com/thessocinco/mars-syntax-app.git)

---

## Technology
- Java
- Android Studio
- SQLite

**Database Tool:** DB Browser for SQLite

**Application Type:** Field Local (Offline) Application

**Deployment Area:** Region 11 & 12

---

## Purpose
Supports small and large booking activities and van selling operations for field personnel

---

## Key Functions
- SMS-based data transmission for sending booking and van selling transaction data from the field to the central system
- GPS-enabled location tracking to capture and verify field transaction locations
- System message handling for operational alerts and updates
- Offline data storage using SQLite, with database management and inspection performed via DB Browser for SQLite

---

## Impact / Results
- Improved field efficiency by enabling real-time booking and van selling reporting
- Enhanced accuracy of transactions through automated data processing
- Strengthened monitoring and coordination of field personnel across assigned regions
- Reliable offline data management with secure local storage and synchronization support
